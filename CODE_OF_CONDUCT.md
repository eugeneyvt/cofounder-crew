# Code of Conduct

Cofounder Crew is a developer tool project. Keep discussion practical, respectful, and focused on improving the software.

## Expected Behavior

- Be direct without being hostile.
- Critique ideas, code, docs, and behavior rather than people.
- Assume good faith until there is clear evidence otherwise.
- Keep issues and pull requests on topic.
- Respect maintainers' time by providing concrete context, reproduction steps, and tradeoffs.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or sustained disruption.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Security disclosure details in public issues before a fix path exists.

## Enforcement

Maintainers may edit, hide, lock, or delete comments; close issues; reject pull requests; or block participants who violate these expectations.

If you need to report a conduct issue, open a GitHub issue with minimal public detail and ask for a private contact path.
